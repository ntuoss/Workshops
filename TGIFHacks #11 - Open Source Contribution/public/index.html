Open Source Contribution†

A Gentle Introduction

Checklist†

  • Get the source
  • Build the development version
  • Find a bug
  • Comment on bug to announce your work
  • Hack away!
  • Test your fix/addition
  • Submit for review

Get the source†

Find the source of the project you want to contribute to. E.g.

Firefox

hg clone https://hg.mozilla.org/mozilla-central

Build the development version†

Follow project's intructions

  • Firefox
  • Python
  • Rust

Find a bug†

This will be the second most time-consuming task. Usually, there are easy bugs for new comers to get started.

  • Firefox
  • Python
  • Rust

Comment on bug to announce your work†

So as not to waste other people's effort. Just say something like this.

  • Hey, is there anyone else working on this bug? If not, I'm interested in solving it.

Hack away!†

Let your inner nerd take over :)

Test your fix/addition†

Just to make sure that you did not break others' awesome code.

Submit for review†

Celebrate :)
